# Neco

NMO Ecosystem.

## What is it?

Neco is a brand new website for NMO.

## Deployment

Need Node.js 22+ and npm 11+.

```bash
git clone https://github.com/nmo-ecosystem/neco.git
cd neco
npm install
npm run dev
```
